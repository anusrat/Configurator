----------------------OWL Configurator------------------------------

The tool provides a GUI to read and modify configurable parameters from XML file.

To run it, the following need to be installed:

1. Python 3.3
2. PyQT4 from http://www.riverbankcomputing.com/software/pyqt/download
3. ElementTree package for Python (shipped with Python 2.7 and higher).
   More documentation for it can be found at:       https://docs.python.org/2/library/xml.etree.elementtree.html

**A sample XML file test.xml is provided. This is the expected structure of the input xml file to be provided to the configurator.

How to run:

**Change into directory where you have cloned the repository and run the following command:
python xml_edit.py
